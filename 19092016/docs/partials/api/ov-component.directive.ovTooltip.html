ovTooltip
directive in module ov-component

Description

Show a tooltip by clicking on an element

Usage

as attribute
<ANY ov-tooltip
     i18n-key="{string}"
     [template-url="{string}"]
     [css-class="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
i18nKeystring

i18n key to be translated and displayed by clicking on an element

templateUrl
(optional)
string

url to the custom template to be loaded and displayed

cssClass
(optional)
string

override class

Example

Basic example: 

<!-- HTML -->
    <i class="fa fa-info-circle" ov-tooltip i18n-key="common.deviceSelection"></i>

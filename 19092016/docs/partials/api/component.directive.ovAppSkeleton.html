ovAppSkeleton
directive in module component

Description

Provide skeleton for new app, It provides sidebar, breadcrumbs and cached the last visited page.

Dependencies

  • $i18next
  • $location
  • ovTaskFlow
  • $routeParams
  • OvTreeModel
  • $log
  • $transition

Usage

as element:
<ov-app-skeleton
       app-id="{String}"
       cached-object="{Object}"
       app-root-url="{String}"
       [redirect-url]="{String}"
       nav-data="{String}"
       side-bar-menu="{String}"
       content-wrapper-css="{String}">
</ov-app-skeleton>
as attribute
<div ov-app-skeleton
     app-id="{String}"
     cached-object="{Object}"
     app-root-url="{String}"
     [redirect-url]="{String}"
     nav-data="{String}"
     side-bar-menu="{String}"
     content-wrapper-css="{String}">
   ...
</div>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
appIdString

ID used to

cachedObjectObject

Used to cached last visited page

appRootUrlString

Root URL

[redirectUrl]String

If child route isn't found in sideBarMenu redirect to this URL

(default: appRootUrl)

navDataString

Breadcrumb model

sideBarMenuString

SideBarMenu Model

contentWrapperCssString

name of wrapper css class name

Example

//tips
   You can refer to these files for real app:
   app/scripts/vxlans/wrapper.html
   app/scripts/vxlans/vxlansWrapperCtrl.js
   app/scripts/vxlans/services/vxlansWrapper.js

//html
   <ov-app-skeleton app-id="vxlans" app-root-url="vm.appRootUrl" redirect-url="vm.redirectUrl"
   cached-object="vm.cachedObject" side-bar-menu="vm.sideBarMenu"
   nav-data="vm.navData" content-wrapper-css="vm.contentWrapperCss"></ov-app-skeleton>

//js
   vm.sideBarMenu = vxlansWrapperService.getSideBarMenuData();
   vm.navData = vxlansWrapperService.getNavData();
   vm.cachedObject = vxlansWrapperService.cachedObject;
   vm.redirectUrl = 'vxlans/vxlan-service';
   vm.appRootUrl = '/vxlans';
   vm.contentWrapperCss = 'vxlans-right-pane';

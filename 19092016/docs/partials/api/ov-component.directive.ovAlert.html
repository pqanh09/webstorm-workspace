ovAlert
directive in module ov-component

Description

Show/Hide alert messages

Usage

as attribute
<ANY ov-alert="{object}"
     alert-id="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
ovAlertobject

show/hide alert.

alertIdstring

An ID to be added to the element to support automation test e.g. alter-id="test"

Example

Basic Example: 

<!-- HTML -->
 <div alert-id="wma-sample-alert" ov-alert="ctrl.alert"></div>

//JS
 //build alert object
   ctrl.alert = ovAlertBuilder.getBuilder().build();
 //show success msg
   ctrl.alert._success('success msg');
 //show error msg
   ctrl.alert._error('error msg');
 //show warning msg
   ctrl.alert._warning('warning msg');
 //show info msg
   ctrl.alert._info('info msg');
 //hide alert
   ctrl.alert._hideAlert();

Config Example: 

<!-- HTML -->
 <div alert-id="wma-sample-alert" ov-alert="ctrl.alert"></div>

//JS
 //config for alert
 var config = {
    isDismissal: true, //show,hide remove alert button
    timeout: 1000 // set timeout to auto remove alert
  };
 //build alert object
 ctrl.alert = ovAlertBuilder.getBuilder().build();
 //show success msg
 ctrl.alert._success('success msg', config);

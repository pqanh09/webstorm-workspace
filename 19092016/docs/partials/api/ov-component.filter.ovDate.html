ovDate
filter in module ov-component

Description

Take uses of formatTime, formatDate, and formatDateTime methods from ovUserPrefSrv service to filter input value.

Dependencies

  • ovUserPrefSrv

Usage

In HTML Template Binding

{{ ovDate_expression | ovDate:type }}

In JavaScript

$filter('ovDate')(input, type)

Parameters

ParamTypeDetails
inputstring

Date value to format

typestring

Formatter type. Available formatter types:

  • time
  • date
  • dateTime (Default)

Example

Basic example: 

  // HTML
  <div>{{ctrl1.ovDate.value | ovDate}}</div>

  // JS
  ctrl1.ovDate = {
    value: new Date()
  };

Giving formatter type example: 

  // HTML
  <div>{{ctrl1.ovDate.value | ovDate:'date'}}</div>

ovAccordion
directive in module ov-component

Description

Accordion component

Usage

as attribute
<ANY ov-accordion
     ov-expanded="{boolean}"
     [on-transition-end="{function}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
ovExpandedboolean

A model to bind the current state to, true if the accordion is expanded

onTransitionEnd
(optional)
function

callback function to be executed when transition end

Example

Basic example: 

<!-- HTML -->

 <div class="ov-accordion">

   <div class="ov-ac-pane">
     <div class="ov-ac-header" ov-expanded="ctrl.accordion.pane1">
      pane header 1
     </div>
     <div class="ov-ac-body">
      pane body 1
     </div>
   </div>

   <div class="ov-ac-pane">
     <div class="ov-ac-header" ov-expanded="ctrl.accordion.pane2">
      pane header 2
     </div>
     <div class="ov-ac-body">
      pane body 2
     </div>
   </div>

 </div>


 //JS
 ctrl.accordion = {
  pane1: true,
  pane2: false
 }

ovUiLog
service in module ov-component

Description

Manage UI activities logs.

Methods

  • clearLog()

    Clear all current ui logs.

  • getLogger(appName)

    Return logger for a specific app

    Parameters
    ParamTypeDetails
    appNameString

    Name of the app to get logger

    Returns
    Object

    logger: a logger have .info() and .error() methods (those functions only work in debug mode).

    • .info(message): write message as info log
    • .error(message): write message as error log

  • getUiLog()

    Return all current ui logs.

    Returns
    Array

    logs: an array contains all current UI logs

Example

Logger for app Discovery 

//JS
var logger = ovUiLogService.getLogger('discovery');
logger.info('this is an information message');
logger.error('this is an error message');

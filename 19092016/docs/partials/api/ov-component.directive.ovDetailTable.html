ovDetailTable
directive in module ov-component

Description

The component help to show data as tabular mode

Dependencies

  • $ovDataViewService

Usage

as attribute
<ANY ov-detail-table
     view-data="{array}"
     grid-columns="{array}">
   ...
</ANY>

Parameters

ParamTypeDetails
viewDataarray

array of data

gridColumnsarray

array of column

Example

//HTML
<div ov-detail-table
         view-data="vm.data"
         grid-columns="vm.columns">
</div>

//In JS
vm.data = [
   {
     key1: 'value1',
     key2: 'value2'
   }
];

var attrs = {
   column1: {
     key: 'key1',
     i18nKey: 'i18nKey1'
   },
  column2: {
    key: 'key2',
    i18nKey: 'i18nKey2'
  }
};
vm.columns = ovDataViewService.getColumnList(attrs);

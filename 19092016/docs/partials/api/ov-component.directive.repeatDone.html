repeatDone
directive in module ov-component

Description

Working together with ngRepeat, repeatDone directive executes given callback function right after finishing repeating. It also adds repeat-done CSS class to the parent element in case user needs to apply custom CSS

Usage

as attribute
<ANY repeat-done="{function}">
   ...
</ANY>

Parameters

ParamTypeDetails
repeatDonefunction

Callback function to execute

Example

Basic example:

  // HTML
  <div>
    <div>{{ctrl1.repeatDone.status}}</div>
    <div ng-repeat="item in ctrl1.repeatDone.itemList track by $index" repeat-done="ctrl1.repeatDone.onDone()">
      {{item.name}}
    </div>
  </div>

  // JS
  ctrl1.repeatDone = {
    status: 'Repeating...',
    itemList: [{name: 'Item 1'}, {name: 'Item 2'}, {name: 'Item 3'}, {name: 'Item 4'}, {name: 'Item 5'}],
    onDone: function () {
      this.status = 'Done repeating';
    }
  };

ovDateTimePicker
directive in module ov-component

Description

This component consists actually of 2 sub component UI widgets one for the date and one for the time selection process. The developers can configure which of those are needed and also the granularity that the component will allow the users to select a date/time. Developers also choose the format that the selected datetime will be displayed in the input field. The component uses on jQuery, moment.js and bootstrap libraries.

Usage

as element:
<ov-date-time-picker
       pick-date="{boolean}"
       pick-date="{boolean}"
       widget-container="{string}"
       chosen-date="{millisecond}"
       initial-date="{millisecond}"
       date-time-format="{string}"
       override-css="{string}">
</ov-date-time-picker>
as attribute
<ANY ov-date-time-picker
     pick-date="{boolean}"
     pick-date="{boolean}"
     widget-container="{string}"
     chosen-date="{millisecond}"
     initial-date="{millisecond}"
     date-time-format="{string}"
     override-css="{string}">
   ...
</ANY>

Example

Minimum Setup 

  <ov-date-time-picker
       chosen-date="vm.chosenDate">
  </ov-date-time-picker>

Show date picker only 

  <ov-date-time-picker
       pick-time="false"
       chosen-date="vm.chosenDate">
  </ov-date-time-picker>

Show time picker only 

  <ov-date-time-picker
       pick-date="false"
       chosen-date="vm.chosenDate">
  </ov-date-time-picker>

ovLineChart
directive in module ov-component

Description

Displays a line graph based on the given data.

Usage

as element:
<ov-line-chart
       chart-data="{Array}"
       chart-config="{Object}"
       [on-line-click:="{function}"]
       [on-drag-left:="{function}"]
       [on-drag-right:="{function}"]>
</ov-line-chart>

Parameters

ParamTypeDetails
chartDataArray

Chart data. Each item in the array represents as a line data

chartConfigObject

Chart config.

  • properties object: properties
    • yTicks (optional) integer: number of ticks on y-axis (default 10).
    • width string or integer : width of the chart by value (px) or percent.
    • height string or integer: height of the chart by value (px) or percent.
    • colors string or array: colors to be used in charts, can be array of color code or d3 colors collection name
    • pointStrokeColor string: stroke color of the data points
    • margin object: define margin for the chart
    • pointRadius integer: radius of each data point
    • gridColor string: color for grid lines
    • showPointsTexts (optional) boolean: show value on each data point (default false)
    • tickFormat (optional) function: a callBack used to format the x-ticks, will be called with (data, index, xIntervalType), should return the formatted string.
    • yLabel (optional) string: label will appear on the y-axis (default null)
    • zoom (optional) boolean: allows zoom in and zoom out the chart using mouse scroll (default false)
  • mapping object: define map values
    • name string: property name can be used to get name of the line
    • xValue string: property name can be used to get x-axis value of the data point
    • yValue string: property name can be used to get y-axis value of the data point
  • label object: define properties for chart's labels
    • color string: label text color
    • fontFamily string: label text font
    • fontSize string: label text font-size
  • multiset (optional) boolean: draw many set of chart (default false)
onLineClick:
(optional)
function

a callback function will be called when click on a line. Line data and index will be passed as arguments

onDragLeft:
(optional)
function

a callback function will be called when click on a label

onDragRight:
(optional)
function

a callback function will be called when click on a label

Example

Basic Example: 

 //HTML
 <ov-line-chart data="ctrl.lineChartData" chart-config="ctrl.lineChartConfig"></ov-line-chart>

//JS
 ctrl.lineChartConfig = {
    properties: {
      xIntervalType: 'hours',
      xInterval: 1,
      yTicks: 5,
      width: '100%',
      height: '100%',
      colors: 'colorScale20'
      pointStrokeColor: '#ffffff',
      margin: {
        top: 15,
        right: 70,
        bottom: 110,
        left: 110
      },
      pointRadius: 3,
      gridColor: '#e5e5e5',
      showPointsTexts: false,
      yLabel: 'Resource Utilization Index',
      zoom: true
    },
    label: {
      color: 'gray',
      fontFamily: 'Arial',
      fontSize: '12px'
    },
    mapping: {
      name: 'displayName',
      xValue: 'time',
      yValue: 'value'
    }
  };

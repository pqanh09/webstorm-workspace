ovNavButton
directive in module ov-component

Description

Render a navigation button under the form of an anchor link. The rendered element contains an arrow icon and followed by a label

Usage

as attribute
<ANY ov-nav-button
     direction="{expression|string}"
     label="{expression|string}">
   ...
</ANY>

Parameters

ParamTypeDetails
directionexpressionstring

Angular expression or string to be evaluated as the direction of arrow icon. Accepted values:

  • left: left navigation arrow <
  • right: left navigation arrow >
labelexpressionstring

Angular expression or string to be evaluated as a label.

Example

Basic example: 

  // HTML
  <div ov-nav-button
       direction="left"
       label="Back">
  </div>

Using expression:

  // HTML
  <div ov-nav-button
       direction="{{ctr1.ovNavButton.direction}}"
       label="{{ctr1.ovNavButton.label}}">
  </div>

  // JS
  ctrl1.ovNavButton = {
    direction: 'left',
    label: 'Back'
  };

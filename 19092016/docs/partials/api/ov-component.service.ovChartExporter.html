ovChartExporter
service in module ov-component

Description

Print or export charts to PDF, PNG.

Methods

  • exportChart(data, options)

    Return logger for a specific app

    Parameters
    ParamTypeDetails
    dataobject

    data for exporting, included: chartData, chartConfig, legendData, legendConfig

    optionsobject

    exporting options, included: title, format, exportChartType

Properties

  • exportChartType

    Enum of export chart type values (to be used in export options): pieChart, stackedBarChart, lineChart

  • exportFormat

    Enum of export format values (to be used in export options): png, print, pdf

Example

Exporting a stackedBarChart 

//JS
var exportData = {
  chartData: stackedBarChartData,
  chartConfig: barChartConfig,
  legendData: stackedLegendData,
  legendConfig: barLegendConfig
};
var exportOption = {
  title: 'This is the title for the exported chart',
  format: ovExportChartService.exportFormat.png,
  exportChartType: ovExportChartService.exportChartType.stackedBarChart
};

ovExportChartService.exportChart(exportData, exportOption);

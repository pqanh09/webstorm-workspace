aDisabled
directive in module ov-component

Description

Enable/disable clicking on the element (usually a anchor link) and change the CSS styles accordingly.

Usage

as attribute
<ANY a-disabled="{boolean}"
     ng-click="{function}">
   ...
</ANY>

Parameters

ParamTypeDetails
aDisabledboolean

Enable/disable clicking on the element

ngClickfunction

Callback function which is allowed/prevented to/from executing when aDisabled is set to true/false.

Example

Basic example: 

  // HTML
  <a a-disabled="ctrl1.aDisabled.disabled"
     ng-click="ctrl1.aDisabled.onClick()">
     Click me
  </a>

  // JS

  ctrl1.aDisabled = {
    disabled: true,
    onClick: function () {
      console.log('Click on an anchor link.');
    }
  };

ovAutoFocus
directive in module ov-component

Description

Initially focus an element if not specified or specified to true. If there are multiple elements which have ovAutoFocus set to true, then the last one will be focused

Usage

as attribute
<ANY ov-auto-focus="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
ovAutoFocus
(optional)
boolean

Initially focused state

Example

Basic example: 

  // HTML
  <input ov-auto-focus>

Use controller's property example: 

  // HTML
  <input ov-auto-focus="ctrl1.ovAutoFocus.focused">

  // JS
  ctrl1.ovAutoFocus = {
    focused: true
  };

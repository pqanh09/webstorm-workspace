ovSelectShowService
service in module ov-component

Description

The service wraps ovSlickDoubleList directive to select show/hide columns/fields of table/list view.

Methods

  • showListSetting(scope, viewConfig, [idApp], [finishCb], [cancelCb])

    This function is a shortcut of "showSelectDlg", it was customized for "ovDataView" to select show/hide fields of list view. Instead of passing 'config', 'sl', 'hl' param, we just need to pass one object 'viewConfig'. Please see viewConfig param and check ovDataview directive for more detail.

    Parameters
    ParamTypeDetails
    scopeobject

    The owner scope which contains table/list.

    viewConfigobject

    This object is viewConfig property of ov-data-view object.

    [idApp]string

    Provide unique id of directive for automation test.

    [finishCb]function

    This function will be called when user clicks on ok button.

    [cancelCb]function

    This function will be called when user clicks on cancel button.

    Returns
    null

    This function does not return.

  • showSelectDlg(scope, config, sl, hl, idApp, finishCb, cancelCb)

    Provide to user a screen to select show/hide columns/fields of table/list view.

    Parameters
    ParamTypeDetails
    scopeobject

    The owner scope which contains table/list.

    configobject

    This is config object:

    • title: title of dialog.
    • itemTemplateUrl: item template url.
    • searchBy: the property items will be searched by.

    In shorthand, it can be empty object, the service will use default config.

    slarray

    An array of show items.

    hlarray

    An array of hide items.

    idAppstring

    Provide unique id of directive for automation test.

    finishCbfunction

    This function will be called when user clicks on ok button.

    cancelCbfunction

    This function will be called when user clicks on cancel button.

    Returns
    null

    This function does not return.

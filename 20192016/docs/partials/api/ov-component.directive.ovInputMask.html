ovInputMask
directive in module ov-component

Description

The ovInputMask limits characters that can be entered into an input field.

Dictates the format an input field value must follow.

Removes or preserves the mask syntax hint on an input field.

Usage

as attribute
<ANY ov-input-mask="{string|object}">
   ...
</ANY>

Parameters

ParamTypeDetails
ovInputMaskstringobject

sets the masks of input field.

  • string: there are two types 'mac' (mac address only) or 'mac' (mac address includes characters).
  • object: config object of jquery-iMask plugin (https://github.com/zyklus/jQuery-iMask).

Example

<!--HTML-->
  <input type="text" class="macAddress form-control"
  ov-input-mask="'mac'" ng-model="classification.ruleValue.macAddress"/>

//JS

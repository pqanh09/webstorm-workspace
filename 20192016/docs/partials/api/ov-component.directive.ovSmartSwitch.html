ovSmartSwitch
directive in module ov-component

Description

The ovSmartSwitch allows user to select anything with two states e.g On vs Off, Disabled vs Enable, True vs False.

Usage

as attribute
<ANY ov-smart-switch
     smart-switch-id="{string}"
     [smart-peer="{boolean}"]
     [label-on="{string}"]
     [label-off="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
smartSwitchIdstring

Provide unique id of directive for automation test.

smartPeer
(optional)
boolean

Set it true if two states is peer.

labelOn
(optional)
string

Label for on state. It can be the i18next key, it does translation itself.

labelOff
(optional)
string

Label for off state. It can be the i18next key, it does translation itself.

Example

<!--HTML-->
  <input type="checkbox" ov-smart-switch smart-switch-id="traps-refresh-interval-switch"
  label-on="'button-label.on'" ng-true-value="true"
  label-off="'button-label.off'" ng-false-value="false"
  ng-model="vm.extendModel.settingModel.trapInterval.enabled">

ovFormatter
directive in module ov-component

Description

The ovFormatter directive validates with RegExp to prevent invalid user input. It also requires ngModel.

Usage

as attribute
<ANY ov-formatter="{string|regEx}"
     ng-model="{any}">
   ...
</ANY>

Parameters

ParamTypeDetails
ovFormatterstringregEx

sets the rules that the model is validated by.

  • string: there are two types 'integer' or 'number'.
  • regEx: patterns used to match character combinations in strings.
ngModelany

Assignable angular expression to format.

Example

Validate with available types: 

 <!--HTML-->
   <input type="text" class="form-control input-field" id="demo-input-field"
   ng-model="vm.inputObject.demoValue" ov-formatter="vm.regex">

//JS
   vm.regex = 'integer';

Validate with RegEx: 

 <!--HTML-->
   <input type="text" class="form-control input-field" id="demo-input-field"
   ng-model="vm.inputObject.demoValue" ov-formatter="vm.regex">

//JS
   vm.regex = /^-?\d*\.?\d*$/;

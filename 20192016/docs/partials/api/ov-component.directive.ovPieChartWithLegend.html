ovPieChartWithLegend
directive in module ov-component

Description

Displays a pieChart based on the given data. Supported drill down, back to parent, save state, export chart.

Usage

as element:
<ov-pie-chart-with-legend
       chart-data="{array}"
       [chart-config="{object}"]
       [on-pie-click="{function}"]
       [state="{function}"]>
</ov-pie-chart-with-legend>
as attribute
<ANY ov-pie-chart-with-legend
     chart-data="{array}"
     [chart-config="{object}"]
     [on-pie-click="{function}"]
     [state="{function}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
chartDataarray

Chart data

chartConfig
(optional)
object

Chart config.

  • pieChartConfig object: Config for the pie-chart, similar to config in pieChart component, have an additional optional property:
    • childrenProps (optional) object: included key and title, in case pie have multiple children to be drilled down.
  • legendConfig object: Config for the legend, similar to config in ovLegend component.
  • isReportMode (optional) object: Define if the chart is drawing in report mode (default false)
  • isDashboard (optional) object: Define if the chart is drawing in dashboard mode (default false)
onPieClick
(optional)
function

callBack function will be called when click on a bar. whole bar data and index will be passed as arguments

state
(optional)
function

state of the current chart, use this to save current chart state.

Example

Basic Example: 

 //HTML
 <ov-pie-chart-with-legend chart-data="ctrl.pieChartData" chart-config="ctrl.pieChartWithLegendConfig"></ov-pie-chart-with-legend>

//JS

 var pieChartConfig = {...};
 var legendConfig = {...};

 ctrl.pieChartWithLegendConfig = {
   pieChartConfig: pieChartConfig,
   legendConfig: legendConfig,
   isReportMode: false,
   isDashboard: false
 };

 ctrl.pieChartData = [...];

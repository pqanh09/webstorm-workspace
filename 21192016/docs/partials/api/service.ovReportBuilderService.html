ovReportBuilderService
service in module service

Description

Service that allows user to add a directive or custom controller/templateUrl to report app

Methods

  • addToReport($scope, reportWidget)

    Parameters
    ParamTypeDetails
    $scopeobject

    $scope object of controller

    reportWidgetobject

    config object

    • title – {string} - Report's title
    • id – {string} - directive tag name ex: ov-chart
    • controller – {string} - controller name
    • templateUrl – {string} - template url

Example

Existing directive and need to add to report: 

  ovReportBuilderService.addToReport($scope, {
     title: 'Report title',
     id: 'directive-tag-name'
  }
Don't want to write a directive but have a controller and a template: 
  ovReportBuilderService.addToReport($scope, {
     title: 'Report title',
     controller: 'yourController',
     templateUrl: 'yourTemplateUrl'
  }

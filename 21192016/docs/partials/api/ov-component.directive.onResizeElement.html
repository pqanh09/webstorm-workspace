onResizeElement
directive in module ov-component

Description

Execute given callback function whenever window element gets re-sized and pass new width value to the callback as a parameter.

Usage

as attribute
<ANY on-resize-element="{function}">
   ...
</ANY>

Parameters

ParamTypeDetails
onResizeElementfunction

Callback function to execute

Example

Basic example: 

  // HTML
  <div ov-resize-element="ctrl1.ovResizeElem.onResize">
  </div>

  // JS
  ctrl1.ovResizeElem = {
    onResize: function (newWidthSize) {
      if (newWidthSize < 768) {
        // Try to collapse something
      } else {
        // Expand them
      }
    }
  };

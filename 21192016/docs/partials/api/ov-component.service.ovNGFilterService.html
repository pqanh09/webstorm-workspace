ovNGFilterService
service in module ov-component

Description

The simple and convenient service of complex datum queries. It is used in ovDataView directive.

Methods

  • doNGFilter(data, filterSelected)

    Do filter on the data by filter list.

    Parameters
    ParamTypeDetails
    dataarray

    Array of data want to filter.

    filterSelectedarray

    Array of filters want to apply. Check "showFilterDialog" for more detail about a filter object format.

    Returns
    array

    Return an array of data was filtered by selected filters.

  • showFilterDialog(scope, filterInfo, filterCallback)

    Manage filters (create/edit/delete) and filter datum base on selected filters.

    Parameters
    ParamTypeDetails
    scopeobject

    The owner scope which calls the service.

    filterInfoobject

    This object contains filter information:

    • filterList: All of the filter was created. if there is no filter, this is empty array.
    • filterSelected: All the selected filters which will be applied to filter data.
    • filterFields: Array of attributes on data object that you want to filter.
    • filterBackupData: The original data.
    • filterData: The data after applying selected filters. If there is no selected filter, it is the same as the original data.
    • singleSelect: Set it true if you want only one filter may be selected and vice versa.
    filterCallbackobject

    This object contains filter callbacks:

    • finishCallback: This function will be called after applying filters on data.
    • onAdded: This function will be called whenever there is one filter was created.
    • onEdited: This function will be called whenever there is one filter was edited.
    • onDeleted: This function will be called whenever there is one filter was deleted.
    Returns
    null

    This function does not return.

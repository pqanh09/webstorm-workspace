ovWizard
directive in module ov-component

Description

The Wizard component will provide a user interface that leads a user through some steps The Wizard component UI will contain these elements:

  • Wizard title area:
    • Wizard title text
    • Wizard title icon
  • Wizard page heading area:
    • Current wizard page title text
    • Current wizard page description text
  • Wizard page icon
    • Wizard page content area:
    • Wizard page content template for custom rendering
  • Button bar area:
    • Navigation buttons (Back, Next) and action buttons (Finish, Cancel) shall be put in here

Dependencies

  • ovValidatorServices
  • ovWizardDataBuilder

Usage

as element:
<ov-wizard
       wz-model="{object}">
</ov-wizard>

Parameters

ParamTypeDetails
wzModelobject

an object that contains configurations for the directive to render.

Example

Basic example: 

  <!-- HTML -->
   <ov-wizard wz-model="ctrl.wizard"></ov-wizard>

   //JS
   ctrl.wizard = ovWizardDataBuilder.getBuilder()
     .setSequencePages([step1, step2, step3])
     .setTitle('Registration')
     .build();
Step model

  • id {string} - (Optional) step id
  • stepName {string} - step name
  • title {string} - step title
  • desc {string} - step description
  • templateUrl {string} - path to the template file to render the page content
  • initValid {function} - function can be executed on init the wizard, return true or false to check whether the wizard can be valid or not, if this page is invalid, you can not move to next step
  • isValid {function} - function can be executed when this is current step, return true or false to check whether the wizard can be valid or not, if this page is invalid, you can not move to next step
  • onInit {function} - (Optional) callback function can be executed one time on init the this step
  • onDisplay {function} - (Optional) callback function can be executed when this is current step
  • onDestroy {function} - (Optional) callback function can be executed when this step is destroyed

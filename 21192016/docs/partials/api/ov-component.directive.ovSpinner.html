ovSpinner
directive in module ov-component

Description

The number spinner component

Usage

as element:
<ov-spinner
       sp-value="{number}"
       [sp-config="{object}"]
       [sp-id="{string}"]
       [sp-disabled="{boolean}"]
       [placeholder="{string}"]>
</ov-spinner>

Parameters

ParamTypeDetails
spValuenumber

A model to bind the value to

spConfig
(optional)
object

config for custom the component

  • min {number} (Optional) - min number (Default -Infinity)
  • max {number} (Optional) - max number (Default Infinity)
  • decimals {number} (Optional) - decimals (Default 0)
  • step {number} (Optional) - step (Default 1)
  • prefix {string} (Optional)- prefix label
  • postfix {string} (Optional)- postfix label
  • emptyAllowed {boolean} (Optional)- allow empty input (Default true)
spId
(optional)
string

An ID to be added to the element to support automation test e.g. sp-id="test"

spDisabled
(optional)
boolean

Determines whether or not to disable the component

placeholder
(optional)
string

Placeholder text that will be forwarded to the input.

Example

Basic example: 

<!-- HTML -->
   <ov-spinner  sp-value="ctrl.value" sp-config="ctrl.config"></ov-spinner>

 //JS
   ctrl.config = {
    min: 0,
    max: 100
   }

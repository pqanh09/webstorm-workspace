ovSvgLegend
directive in module ov-component

Description

Create chart legends using SVG for exporting purpose.

Usage

as element:
<ov-svg-legend
       legend-data="{Array}"
       [legend-config="{Object}"]>
</ov-svg-legend>
as attribute
<ANY ov-svg-legend
     legend-data="{Array}"
     [legend-config="{Object}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
legendDataArray

legend data

legendConfig
(optional)
Object

Legend config.

  • width Integer (optional): Legends container width (default 600)
  • fixedColors Object (optional): A mapper to define fixed color for some data items.
  • mapping (optional) object: define map values
    • name string: property name can be used to get legend item name (default name)
    • value string: property name can be used to get legend item value (default value)

Example

Basic Example: 

 //HTML
 <ov-svg-legend legend-config="ctrl.legendConfig"
 legend-data="ctrl.legendData">
 </ov-svg-legend>

//JS
 ctrl.legendConfig = {
    width: 600px,
    mapping: {
      name: 'displayName',
      value: 'count'
    },
    fixedColors: {
      Others: '#808080',
      OTHERS: '#808080'
    }
  };

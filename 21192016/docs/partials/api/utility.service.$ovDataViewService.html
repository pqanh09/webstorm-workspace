$ovDataViewService
service in module utility

Description

Service of ovDataView component

Methods

  • getColumnList(attrs)

    Generate column for view components (ovDataView, ovNgListBox, ovDetailTable...)

    Parameters
    ParamTypeDetails
    attrsobject

    columns object 

    Ex:
{
   column1: {
       key1: 'key1',
       i18nKey: 'i18nKey1'
   }
}
    Returns
    array

    column list

Example

Get column list: 

   //IN JS
   var attrs = {
       column1: {
           key: 'key1',
           i18nKey: 'i18nKey1'
       },
       column2: {
           key: 'key2',
           i18nKey: 'i18nKey2'
       }
   };
   $ovDataViewService.getColumnList(attrs);

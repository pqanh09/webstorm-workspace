autoExec
directive in module ov-component

Description

Execute expected expression when width value of the view port is less than or equal to specified value

Usage

as attribute
<ANY auto-exec="{expression}"
     when-width="{number}">
   ...
</ANY>

Parameters

ParamTypeDetails
autoExecexpression

The expression to be executed

whenWidthnumber

Width value in pixel

Example

Basic example: 

  // HTML
  <div auto-exec="state.menuExpanded = false"
       when-width="768">
  </div>

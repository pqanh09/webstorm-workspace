ovAutoComplete
directive in module ov-component

Description

ov-auto-complete is a special input component with a drop-down of all possible matches to a custom query. This component allows you to provide real-time suggestions as the user types in the input area

Usage

as element:
<ov-auto-complete
       ng-model="{string}"
       ov-ng-options="{string}"
       [placeholder="{string}"]
       [id="{string}"]
       [ng-change="{expression}"]
       [ng-disabled="{expression}"]
       [ov-auto-focus="{expression}"]>
</ov-auto-complete>

Parameters

ParamTypeDetails
ngModelstring

Assignable angular expression to data-bind to

ovNgOptionsstring

sets the options that the select is populated with and defines what is set on the model on selection.

  • modelValue for item in collection
  • modelValue as label for item in collection
placeholder
(optional)
string

placeholder of input.

id
(optional)
string

An ID to be added to the element to support automation test e.g. id="test"

ngChange
(optional)
expression

Angular expression to be executed when selected option(s) changes due to user interaction with the select element.

ngDisabled
(optional)
expression

Determines whether or not to disable the component

ovAutoFocus
(optional)
expression

Set the focus to the element in HTML form

Example

Basic Example: 

 //HTML
 <ov-auto-complete
 ng-model="ctrl.model"
 ov-ng-options="item.value as item.name for item in ctrl.items">
 </ov-auto-complete>

//JS
 ctrl.items = [
 {name: 'One', value: 1},
 {name: 'Two', value: 2}
 ];
 ctrl.model = 1;

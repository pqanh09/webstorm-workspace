ovFocus
directive in module ov-component

Description

Initially focus an element if specify to true. If there are multiple elements which have ovAutoFocus set to true, then the last one will be focused

Usage

as attribute
<ANY ov-focus="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
ovFocusboolean

Initially focused state

Example

Basic example: 

  // HTML
  <input ov-focus="true">

Use controller's property example: 

  // HTML
  <input ov-focus="ctrl1.ovFocus.focused">

  // JS
  ctrl1.ovFocus = {
    focused: true
  };

ovUxSelect
directive in module ov-component

Description

Displays a select box with single and multiple selection, bound to an ng-model.

Usage

as element:
<ov-ux-select
       ng-model="{string}"
       ov-ng-options="{string}"
       [ov-select-setting="{object}"]
       [select-none="{string}"]
       [ux-select-id="{string}"]
       [ng-change="{expression}"]
       [disabled]
       [ov-auto-focus="{expression}"]>
</ov-ux-select>

Parameters

ParamTypeDetails
ngModelstring

Assignable angular expression to data-bind to

ovNgOptionsstring

sets the options that the select is populated with and defines what is set on the model on selection.

  • modelValue for item in collection
  • modelValue as label for item in collection
ovSelectSetting
(optional)
object

setting for custom select box

  • search(optional) boolean: show/hide search box (default false)
  • sort (optional) boolean: show/hide sort button (default false)
  • multiple (optional) boolean: enable/disable multiple selection (default false)
  • btnCheckAll (optional) boolean: show/hide check all, un-check all buttons, only work with multiple selection (default false)
  • selectUrl (optional) string: custom select template url
  • dropDownMenuUrl (optional): string: custom dropdown template url
  • itemTemplate (optional) string: custom item template url
selectNone
(optional)
string

Add an item with undefined value in top of the dropdown list e.g. select-none="[None]".

uxSelectId
(optional)
string

An ID to be added to the element to support automation test e.g. ux-select-id="test"

ngChange
(optional)
expression

Angular expression to be executed when selected option(s) changes due to user interaction with the select element.

disabled
(optional)
expression

Determines whether or not to disable the component

ovAutoFocus
(optional)
expression

Set the focus to the element in HTML form

Example

Basic Example: 

 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="item.value as item.name for item in ctrl.items">
 </ov-ux-select>

//JS
 ctrl.items = [
 {name: 'One', value: 1},
 {name: 'Two', value: 2}
 ];
 ctrl.selected = 1;

Disable select box Example: 

 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="item.value as item.name for item in ctrl.items"
 disabled="ctrl.disableSelect">
 </ov-ux-select>

//JS
 ctrl.items = [
 {name: 'One', value: 1},
 {name: 'Two', value: 2}
 ];
 ctrl.selected = 1;
 ctrl.disableSelect = true;

Disable a item Example: 

//JS
 ctrl.items = [
 {name: 'One', value: 1, disabled: true},
 {name: 'Two', value: 2}
 ];

ov-ng-options Example: 

 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="item as item.name for item in ctrl.items">
 </ov-ux-select>
 //ctrl.selected will be a object
Array 
 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="value for value in ctrl.items">
 </ov-ux-select>

//JS
 ctrl.items = [1, 2];
 ctrl.selected = 1;

select-none Example: 

 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="item as item.name for item in ctrl.items"
 select-none="[None]">
 </ov-ux-select>

Search, Sort Example: 

 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="item.value as item.name for item in ctrl.items"
 ov-select-setting="ctrl.selectSetting">
 </ov-ux-select>

//JS
 ctrl.items = [
  {name: 'One', value: 1},
  {name: 'Two', value: 2}
 ];
 ctrl.selected = 1;
 ctrl.selectSetting = {
  search: true,
  sort: true
 }

Multiple selection Example: 

 //HTML
 <ov-ux-select
 ng-model="ctrl.selected"
 ov-ng-options="item.value as item.name for item in ctrl.items"
 ov-select-setting="ctrl.selectSetting">
 </ov-ux-select>

//JS
 ctrl.items = [
 {name: 'One', value: 1},
 {name: 'Two', value: 2}
 ];
 ctrl.selected = [1];
 ctrl.selectSetting = {
   multiple:true,
   btnCheckAll:true
 }

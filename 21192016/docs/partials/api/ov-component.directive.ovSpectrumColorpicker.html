ovSpectrumColorpicker
directive in module ov-component

Description

A Color picker component based on spectrum color picker http://bgrins.github.io/spectrum/

Usage

as element:
<ov-spectrum-colorpicker
       ng-model="{string}"
       [options="{object}"]>
</ov-spectrum-colorpicker>

Parameters

ParamTypeDetails
ngModelstring

A model to bind the selected color to

options
(optional)
object

Customize color picker with spectrum params via the options attribute http://bgrins.github.io/spectrum/#options

Example

Basic example: 

<!-- HTML -->
 <ov-spectrum-colorpicker  ng-model="ctrl.color"></ov-spectrum-colorpicker>

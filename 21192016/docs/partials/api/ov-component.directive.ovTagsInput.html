ovTagsInput
directive in module ov-component

Description

ov-tags-input is an input component for building lists of strings or objects. The list items are displayed as 'tags' or 'list'.

Usage

as element:
<ov-tags-input
       ov-ng-options="{string}"
       ng-model="{string}"
       [model-change="{expression}"]
       [current-model="{string}"]
       [current-model-change="{expression}"]
       [placeholder="{string}"]
       [id="{string}"]
       [ng-disabled="{expression}"]
       [config="{object}"]>
</ov-tags-input>

Parameters

ParamTypeDetails
ovNgOptionsstring

sets the options that the select is populated with and defines what is set on the model on selection.

  • modelValue for item in collection
  • modelValue as label for item in collection
ngModelstring

A model to bind the selected item to

modelChange
(optional)
expression

Angular expression to be executed when ngModel updates

currentModel
(optional)
string

A model to bind the input text to

currentModelChange
(optional)
expression

An expression to be run each time the input text updates

placeholder
(optional)
string

Placeholder text that will be forwarded to the input.

id
(optional)
string

An ID to be added to the element to support automation test e.g. id="test"

ngDisabled
(optional)
expression

Determines whether or not to disable the input field

config
(optional)
object

config for custom tags input

  • tagClass(optional) string: bootstrap label class for tags (default label-primary)
  • max (optional) number: maximum number of tags can input (default undefined)
  • maxMessage (optional) string: msg for max tags (default "Cannot input more")
  • allowDuplicate (optional) boolean: allow duplicate (default false)
  • duplicateError (optional) string: error msg for duplicate error(default "The item already exists")

Example

Basic Example: 

 //HTML
 <ov-tags-input
    ov-ng-options="item as item.name for item in ctrl.items"
    ng-model="ctrl.model">
 </ov-tags-input>

//JS
 ctrl.items = [
 {name: 'One', value: 1},
 {name: 'Two', value: 2}
 ];
 ctrl.model = 1;

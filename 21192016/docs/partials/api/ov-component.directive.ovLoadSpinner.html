ovLoadSpinner
directive in module ov-component

Description

Render an circular loading spinner.

Usage

as element:
<ov-load-spinner
       [override-class="{string}"]
       [override-style="{string}"]>
</ov-load-spinner>

Parameters

ParamTypeDetails
overrideClass
(optional)
string

Custom CSS class to wrap the entire element

overrideStyle
(optional)
string

Custom style to wrap the entire element

Example

Basic example: 

  // HTML
  <div>
    <ov-load-spinner></ov-load-spinner>
  </div>

ovLegend
directive in module ov-component

Description

Create interactive chart legends to be used along side with charts in apps.

Usage

as element:
<ov-legend
       legend-data="{array}"
       legend-config="{object}"
       on-click="{function}">
</ov-legend>
as attribute
<ANY ov-legend
     legend-data="{array}"
     legend-config="{object}"
     on-click="{function}">
   ...
</ANY>

Parameters

ParamTypeDetails
legendDataarray

legend data

legendConfigobject

Legend config.

  • width integer (optional): Legends container width (default 100%)
  • height integer (optional): Height of each legend item (default 28)
  • spacing integer (optional): Space between each legend item (default 3)
  • radius integer (optional): Legend item border radius (default 3)
  • textStyle object (optional): Style for legend text (default {'font': '12px Arial'})
  • colors string or Array (optional): colors to be used in charts, can be array of color code or d3 colors collection name (default colorScale20).
  • fixedColors object (optional): A mapper to define fixed color for some data items.
  • multiSelect object (optional): Allows select multiple legend items.
  • showTooltip boolean (optional): Show tooltip on click legend item (default true)
  • hidePercentage boolean (optional): Hide percentage value in the tooltip (default false)
  • mapping (optional) object: define map values
    • name string: property name can be used to get legend item name
    • value string: property name can be used to get legend item value
onClickfunction

a callback that will be called when user click on a legend item. 3 arguments will be passed:

  • data object: data of the legend item
  • index integer: index of the legend item
  • status string: Current status of legend item. Have four possible values: 'in' (click inside legend item), 'out' (click outside legend item), 'tooltip' (showing tooltip), 'toogle' (toogle select legend item, only for multiple select).

Example

Basic Example: 

 //HTML
 <ov-legend legend-config="ctrl.legendConfig" legend-data="ctrl.legendData">
 </ov-legend>

//JS
 ctrl.legendConfig = {
    width: '100%',
    height: 28,
    spacing: 3,
    radius: 3,
    textStyle: {'font': '12px Arial'},
    textColor: 'white',
    colors: 'colorScale20',
    multiSelect: true,
    showTooltip: true,
    hidePercentage: true,
    mapping: {
      name: 'displayName',
      value: 'count'
    },
    fixedColors: {
      Others: '#808080',
      OTHERS: '#808080'
    }
  };
 ctrl.legendData = [

 },

ovStackedBarChart
directive in module ov-component

Description

Displays a stackedBarChart based on the given data.

Usage

as element:
<ov-stacked-bar-chart
       chart-data="{Array}"
       chart-config="{Object}"
       [on-bar-click:="{Function}"]
       [on-partial-bar-click:="{Function}"]
       [on-label-click:="{Function}"]
       [legend-data:="{Array}"]>
</ov-stacked-bar-chart>
as attribute
<ANY ov-stacked-bar-chart
     chart-data="{Array}"
     chart-config="{Object}"
     [on-bar-click:="{Function}"]
     [on-partial-bar-click:="{Function}"]
     [on-label-click:="{Function}"]
     [legend-data:="{Array}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
chartDataArray

Chart data

chartConfigObject

Chart config.

  • properties (optional) object: properties
    • barGap (optional) integer: gap between each bar (default 10)
    • xIntervalType (optional) string: possible values: 'hours', 'days', 'weeks', 'months', 'mondays' (default hours)
    • xInterval (optional) integer: interval - the gap between each tick on x-axis (default 1)
    • colors string or array: colors to be used in charts, can be array of color code or d3 colors collection name
    • margin (optional) object: define margin for the chart (default {top: 10, right: 75, bottom: 85, left: 85})
    • duration (optional) integer: transition duration (default 500)
    • yLabel (optional) string: Label of Y-axis
  • mapping object: define map values
    • name string: property name can be used to get bar-name (default name)
    • value string: property name can be used to get bar-value (default value)
    • tooltipValue (optional) string: property name can be used to get tooltip value. If value of given property name is undefined then percentage value will be displayed
  • label (optional) object: define properties for chart's labels
    • color string: label text color
    • fontFamily string: label text font
    • fontSize string: label text font-size
  • advanced (optional) object: some advanced options for chart
    • noYAxis (optional) boolean: set to true if you want to remove y-axis (default false)
    • noYLabel (optional) boolean: set to true if you want to remove all labels on Y-axis (default false)
  • noAnimation (optional) boolean: set to true if you do not want chart drawing animation (default false)
onBarClick:
(optional)
Function

callBack function will be called when click on a bar. whole bar data and index will be passed as arguments

onPartialBarClick:
(optional)
Function

callBack function will be called when click on a bar. Only partial bar data and index will be passed as arguments

onLabelClick:
(optional)
Function

callBack function will be called when click on a label. Value of the label and index will be passed as arguments

legendData:
(optional)
Array

An array of mapper of barName and color for that bar. E.g. [{ name: 'bar-name', color: '#ddd' }]

Example

Basic Example:

 <ov-stacked-bar-chart chart-data="ctrl.stackedBarChartData" chart-config="ctrl.stackedBarChartConfig"></ov-stacked-bar-chart>

//JS
 ctrl.stackedBarChartConfig = {
    properties: {
      xIntervalType: 'hours',
      xInterval: 1,
      colors: 'colorScale20',
      hoverColor: '',
      margin: {
        top: 20,
        right: 70,
        bottom: 110,
        left: 110
      },
      yLabel: 'yLabel Text goes here'
    },
    mapping: {
      name: 'displayName',
      value: 'displayValue',
      toolTipValue: 'count'
    },
    label: {
      color: 'gray',
      fontFamily: 'sans-serif',
      fontSize: '11px'
    }
  };

ovResize
directive in module ov-component

Description

Automatically resize the height of a element using this directive. Given value is the height value counted from the top of the window. The remaining height of the element will be the height of the window after being minus by given value in REM unit. Min height will be 30rem. The calculation is only executed one time

Usage

as attribute
<ANY ov-resize="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
ovResize
(optional)
string

Height value to resize. Default to 30rem if not specified

Example

Basic example: 

  // HTML - Height of following div will be the height of the window minus 100
  <div ov-resize="100">
  </div>

Use controller's property example: 

  // HTML - Height of following div will be the height of the window minus 100
  <div ov-resize="{{ctrl1.ovResizeValue}}">
  </div>

  // JS
  ctrl1.ovResizeValue = 100;
